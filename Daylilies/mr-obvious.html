Daylily description

The daylily Mr Obvious was hybridized by Gossard in 2008. Mr Obvious was hybridized by crossing (((Swallow Tail Kite x Forbidden Games) x Webster's Pink Wonder) x (Tet. Dallas Star x Webster's Pink Wonder)). The color of Mr Obvious can be described as cream pink self above small light cream green throat. It is a Tetraploid. It has a Unusual FormCrispate form. It has a bud count of 15. It blooms in (Midseason). It's flowers measure (9.5 in, (24 cm)). It has 3 branches. It has a normal scape height of (43 in, (109 cm)). It is Fragrant.
