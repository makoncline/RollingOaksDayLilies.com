Daylily description

The daylily Spacecoast Casino Red was hybridized by Kinnebrew-J. in 2010. Spacecoast Casino Red was hybridized by crossing (Spacecoast Blood Diamond x sdlg). The color of Spacecoast Casino Red can be described as burgundy red with heavy gold edge. It is a Tetraploid. It has a bud count of 25. It blooms in (Early-Midseason,Rebloom). It's flowers measure (6 in, (15 cm)). It has 4 branches. It has a normal scape height of (28 in, (71 cm)).
