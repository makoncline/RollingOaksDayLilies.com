Spacecoast Chomp! Chomp!

Daylily description

The daylily Spacecoast Chomp! Chomp! was hybridized by Kinnebrew-J. in 2011. Spacecoast Chomp! Chomp! was hybridized by crossing (Spacecoast Loose Tooth x Heman). The color of Spacecoast Chomp! Chomp! can be described as dark red with white toothy edge. It is a Tetraploid. It has a bud count of 18. It blooms in (Early,Rebloom). It's flowers measure (10 in, (25 cm)). It has 3 branches. It has a normal scape height of (36 in, (91 cm)).
